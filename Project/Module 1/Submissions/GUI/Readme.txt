Instructions for running GUI
1) Please set path to the folders Chan-Vese and Kernel_GraphCuts
2)Run the GUI
3)Final folder contains data.Select input Data from Final/Dataset1/Reference.
4)Choose Normalization Method as colormap normalization
5)Then select segmentation method from drop down list
*For supervised, it will prompt you to select the classifier data which is stored in Final/Dataset1/.
For example: For classifier trained on 90% of the ground truth, select LDA_classifiers90
6)For metric evaluation, you can select the different metrics from the drop down list.
The segmentation step might take a few seconds
